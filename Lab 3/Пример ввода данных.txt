--- Программа для минимизации взяток для получения лицензии ---

Введите количество чиновников N (N < 100): 3

--- Ввод иерархии ---
Для каждого чиновника от 2 до N введите номер его непосредственного начальника.
Чиновник 1 - корень (начальник всех), для него начальника вводить не нужно.

Начальник чиновника 2: 1
Начальник чиновника 3: 1

--- Ввод вариантов виз и взяток ---
Для каждого чиновника (от 1 до N) введите:
- Количество наборов виз (K, 1..15)
- Для каждого набора: количество виз, затем список чиновников (непосредственные подчиненные), затем стоимость взятки
Если набор пустой (чиновник не требует виз), введите 0 и затем стоимость


Чиновник 1:
  Количество наборов виз: 1
  Набор 1:
    Количество виз (непосредственных подчиненных): 2
    Введите номера 2 чиновников (непосредственных подчиненных): 2 3
    Стоимость взятки: 1200

Чиновник 2:
  Количество наборов виз: 1
  Набор 1:
    Количество виз (непосредственных подчиненных): 0
    Стоимость взятки: 560

Чиновник 3:
  Количество наборов виз: 1
  Набор 1:
    Количество виз (непосредственных подчиненных): 0
    Стоимость взятки: 340

--- Результат ---
Минимальная сумма взяток: 2100 долларов
